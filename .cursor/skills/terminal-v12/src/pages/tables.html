USAGE GUIDELINES

  • Table Wrapper: Always wrap tables in .table-wrapper for proper border-radius and overflow
  • Headers: Use uppercase text in th elements for consistency
  • Hover States: All table rows have v12.0 Spring hover animation
  • Striped Variant: Use .table--striped for alternating row colors
  • Compact Variant: Use .table--compact for dense data with reduced padding
  • Cell Modifiers: .table__mono for code, .table__muted for secondary info
  • Checkboxes: Use .table-check component with custom styling
  • Actions: Place action buttons in .table-actions container
  • User Display: Use .table-user for avatar + info combination
  • Expandable Rows: Toggle .expanded class on row, .visible on content row
  • Progress Bars: Use .table-progress with color variants (success, warning, error)
  • Rich Content: Combine .table-product with .table-image for product tables
  • Grouped Headers: Use colspan/rowspan for multi-level headers
  • Responsive: Tables scroll horizontally on mobile (min-width: 600px)
  • Status Badges: Combine with badge components for status indicators
