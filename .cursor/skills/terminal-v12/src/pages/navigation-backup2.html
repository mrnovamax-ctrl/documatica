NAVIGATION COMPONENTS

BASE CLASS

.topbar - Main navbar container with blur background

MODIFIERS

  • .topbar--dark - dark theme with animated dot pattern
  • .topbar--fixed - sticky position at top of viewport
  • .topbar--logo-center - centered logo with balanced menu layout

ELEMENTS

  • .topbar__inner - flex container for navbar content
  • .topbar__logo - brand logo with mark and text
  • .topbar__menu - navigation links container
  • .topbar__link - individual navigation link
  • .topbar__link--active - active state for current page
  • .topbar__actions - right-aligned action buttons
  • .topbar__btn - secondary button style
  • .topbar__btn-primary - primary button style
  • .topbar__avatar - user avatar circle
  • .topbar__search - expandable search input
  • .topbar__dropdown - dropdown menu container
  • .topbar__dropdown-menu - dropdown content
  • .topbar__dropdown-item - dropdown link
  • .topbar__mega - mega menu container
  • .topbar__mega-menu - mega menu dropdown panel
  • .topbar__mega-grid - 3-column grid layout
  • .topbar__mega-section - column section
  • .topbar__mega-title - section heading
  • .topbar__mega-link - mega menu link with icon
  • .topbar__mega-link-icon - icon container
  • .topbar__mega-link-content - text content wrapper
  • .topbar__mega-link-title - link title
  • .topbar__mega-link-desc - link description

USAGE RULES

  • All components defined in core.css
  • No inline styles allowed
  • Use v12.0 Spring easing: cubic-bezier(0.34, 1.56, 0.64, 1)
  • Combine modifiers: topbar topbar--dark topbar--fixed
  • Responsive: menu hides on mobile (max-width: 1023px)
