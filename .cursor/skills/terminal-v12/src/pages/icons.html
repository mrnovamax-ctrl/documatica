Rule

Icons must have clear visual feedback for all states: Default (subtle), Hover (lift + color), Active (press down), Disabled (low opacity).

SIZE SCALE

Rule

Use standard scale (16px, 20px, 24px, 32px). Icon size should match component size proportionally.

Usage Principles

How to use icons correctly in components

✓ DO

  • Always use stroke-width="1.5"
  • Add stroke-linecap="round" stroke-linejoin="round"
  • Match icon size to component size
  • Use subtle colors in default state
  • Provide visual feedback on hover/active
  • Use semantic colors for status icons
  • Center icons in containers
  • Maintain consistent spacing around icons

✗ DON'T

  • Use stroke-width="2" or higher (too heavy)
  • Mix different stroke widths
  • Use random icon sizes
  • Make default state too bright
  • Skip hover/active states
  • Use wrong colors for status
  • Misalign icons in containers
  • Overcrowd icons with text
