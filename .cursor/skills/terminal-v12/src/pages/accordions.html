01. BASIC ACCORDION

Standard accordion with chevron icons and smooth expansion

Documatica v12.0 is a comprehensive design system built for production deployment across 100+ projects. It features a complete component library with strict quality standards.

The system includes navigation, modals, tabs, accordions, forms, cards, and more - all with v12.0 Spring physics animations and responsive design.

Getting started is simple:

  1. Link the core.css stylesheet to your HTML
  2. Use BEM naming conventions for all components
  3. Follow the spacing scale (spacing-1 through spacing-24)
  4. Use CSS variables for colors and measurements

All modern browsers are fully supported including Chrome, Firefox, Safari, and Edge. The system is built with progressive enhancement and works on all devices from mobile to 4K displays.

Yes! The entire design system is built with CSS variables, making customization straightforward.

Modify colors, spacing, typography, and animations by updating the CSS variables in core.css. All components will automatically adapt to your custom values.
