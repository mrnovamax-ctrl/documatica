RATING SYSTEM RULES

BASE CLASSES

  • .rating - Star rating container
  • .rating-star - Individual star element
  • .rating-star--filled - Filled star (gold)
  • .rating-star--half - Half-filled star

SIZE MODIFIERS

  • .rating--sm - Small (16px)
  • .rating--md - Medium (20px)
  • Base - Default (24px)
  • .rating--lg - Large (32px)
  • .rating--xl - Extra large (48px)

COLOR VARIANTS

  • Default - Gold (#fbbf24)
  • .rating--blue - Brand blue
  • .rating--success - Green
  • .rating--error - Red

COMPONENTS

  • .rating-display - Rating with value text
  • .rating-card - Card with rating and metadata
  • .rating-breakdown - Detailed distribution chart
  • .ai-rating-card - AI confidence score card
  • .rating--interactive - Clickable rating

USAGE RULES

  • All components defined in core.css
  • No inline styles (except dynamic widths)
  • Use v12.0 Spring easing: cubic-bezier(0.34, 1.56, 0.64, 1)
  • Star character: ★ (U+2605)
  • Half stars use gradient mask
  • Interactive ratings have hover/active states
  • All text uppercase with letter-spacing
  • Responsive on all 4 breakpoints
