USAGE GUIDELINES

  • Semantic Colors: Use success (green), warning (yellow), danger (red), info (blue) for appropriate contexts
  • Text Content: Keep badge text short and uppercase (2-12 characters recommended)
  • Smart Badge: Use .badge--smart only for AI-powered or premium features with gradient effect
  • Dot Indicators: Use pulsing dot (.badge__dot--pulse) for active processing states
  • Sizes: Small for compact spaces, default for general use, large for emphasis
  • Icons: 14px SVG icons only, ensure proper stroke-width for clarity
  • Counter Badges: For notification counts and numeric indicators
  • Tags: Use removable tags for filters and selected items with animated removal
  • Category Tags: Use for document classification with colored dot system
  • Label Badges: Use for metadata display (key-value pairs like version, status, license)
  • Avatar Badges: Online (green), busy (red), away (gold), offline (gray)
  • Tag Animation: Click remove button triggers v12.0 Spring physics (scale + rotate + fade)

COMPONENT CLASSES

Base Classes

  • .badge - Base badge component
  • .badge--sm - Small size (10px font)
  • .badge--lg - Large size (14px font)
  • .badge--pill - Fully rounded pill shape
  • .badge--icon - With inline icon
  • .badge--counter - Circular counter badge

Color Variants

  • .badge--success - Green success state
  • .badge--warning - Yellow warning state
  • .badge--danger - Red danger state
  • .badge--info - Blue info state
  • .badge--neutral - Gray neutral state
  • .badge--primary - Brand blue
  • .badge--smart - Gold gradient AI badge
  • .badge--dark - Dark premium badge

Outline Variants

  • .badge--outline - Base outline modifier
  • .badge--outline-primary - Blue outline
  • .badge--outline-success - Green outline
  • .badge--outline-warning - Yellow outline
  • .badge--outline-danger - Red outline
  • .badge--outline-neutral - Gray outline

Advanced Components

  • .badge--dot - Badge with dot indicator
  • .badge__dot - Dot element
  • .badge__dot--pulse - Animated pulsing dot
  • .tag - Removable tag component
  • .tag--primary - Primary tag variant
  • .tag__remove - Remove button
  • .category-tag - Category tag with dot
  • .category-tag__dot - Colored dot (blue/green/gold/red)
  • .label-badge - Key-value badge container
  • .label-badge__key - Badge key (dark)
  • .label-badge__value - Badge value (light/colored)
  • .avatar-badge - Avatar status indicator
  • .avatar-badge--online - Online status (green)
  • .avatar-badge--busy - Busy status (red)
  • .avatar-badge--away - Away status (gold)
  • .avatar-badge--offline - Offline status (gray)
