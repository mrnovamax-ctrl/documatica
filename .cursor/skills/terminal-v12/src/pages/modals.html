MODAL DIALOGS

Modal windows with v12.0 Spring animations, backdrop blur, multiple sizes and variants

01. BASIC MODALS & SIZES

Standard modals with header, body, footer. Sizes: sm (400px), md (600px), lg (800px), xl (1000px)

02. CONFIRMATION MODALS

Modals for user confirmations with icons and centered content

03. FORM MODALS

Modals with forms and inputs

SMALL MODAL

This is a small modal dialog (400px width). Perfect for quick confirmations and simple messages.

MEDIUM MODAL

This is a medium modal dialog (600px width). The default size for most use cases.

It provides enough space for forms, detailed content, and longer messages while remaining compact and focused.

LARGE MODAL

This is a large modal dialog (800px width). Great for complex forms, detailed content, or when you need more horizontal space.

Use this size when displaying tables, multi-column layouts, or extensive information that benefits from additional width.

The content area can accommodate rich media, multiple form sections, or side-by-side comparisons.

EXTRA LARGE MODAL

This is an extra large modal dialog (1000px width). Use for complex workflows, dashboards, or content-heavy interfaces.

This size is ideal for displaying detailed analytics, comprehensive forms with multiple sections, or embedded content like videos and presentations.

LEFT COLUMN

Example content in a two-column layout. Perfect for comparison views or split interfaces.

RIGHT COLUMN

Additional content demonstrating the available space in an XL modal.

FULLSCREEN MODAL

This modal takes up the entire viewport. Perfect for immersive experiences, complex workflows, or when you need maximum screen space.

Use fullscreen modals for:

  • Image or video viewers
  • Document editors
  • Complex multi-step forms
  • Data visualization dashboards

DARK MODE

Dark modal with dotted pattern background. Matches the dark navbar theme from Documatica v12.0.

Uses white text on dark slate background for excellent readability and modern aesthetics.
