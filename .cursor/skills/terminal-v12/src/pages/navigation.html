NAVIGATION SYSTEM

BASE CLASS

.navbar - Responsive navbar with mobile menu

MODIFIERS

  • .navbar--dark - dark theme with dotted pattern background

ELEMENTS

  • .navbar-inner - flex container for navbar content
  • .navbar-brand - brand logo container
  • .brand-text - brand name text
  • .navbar-menu - navigation links (hidden on mobile)
  • .nav-link - navigation link
  • .navbar-actions - action buttons and search
  • .navbar-search - expandable search input
  • .navbar-search-btn - search button
  • .navbar-search-input - search input field
  • .navbar-avatar - user avatar circle
  • .navbar-toggle - mobile menu burger button
  • .navbar-toggle-icon - hamburger icon with animation
  • .navbar-mobile-menu - mobile dropdown menu
  • .navbar-mobile-actions - button in mobile menu
  • .navbar-mobile-nav - mobile navigation list
  • .navbar-mobile-nav-item - mobile nav item
  • .navbar-mobile-nav-link - mobile nav link

USAGE RULES

  • All components defined in core.css
  • No inline styles allowed
  • Use v12.0 Spring easing: cubic-bezier(0.34, 1.56, 0.64, 1)
  • Responsive: burger menu appears on mobile (< 1024px)
  • Dark variant: white text, gold hover, dotted pattern
  • Mobile menu: animated slideDown, full-width actions
