ðŸ’¡ Hero Usage Rules

  • Default Hero - for regular documentation and content pages
  • Large Hero - for homepage and landing pages with CTA buttons
  • Compact Hero - for internal pages with lots of content
  • Hero with Pattern - for premium sections and accent pages
  • Dark Hero - for enterprise sections, use .pattern-dots-dark from background system
  • Hero with Meta - for blogs, articles, news
  • Full Width Hero - place before main-content for edge-to-edge layout

Modifiers

  • .hero--large - increased size
  • .hero--compact - compact size
  • .hero--pattern - with background pattern (+ pattern class)

Backgrounds from system

  • .bg-white, .bg-surface, .bg-surface-light
  • .bg-dark, .bg-primary, .bg-gold
  • .pattern-dots-light, .pattern-dots-dark
  • .pattern-radial-blue, .pattern-radial-gold
