SPACING SYSTEM

The "Air System" - breathing room rules for consistent vertical rhythm

Proper spacing creates visual hierarchy and "breathing room" in design. Documatica v12.0 uses a 4px-based system for all spacing values.

Spacing Scale

Base Unit: 4px

All spacing is a multiple of 4px. This creates a consistent system and simplifies calculations.

Scale Values

  • 1 = 4px = 0.25rem
  • 2 = 8px = 0.5rem
  • 3 = 12px = 0.75rem
  • 4 = 16px = 1rem
  • 6 = 24px = 1.5rem
  • 8 = 32px = 2rem
  • 12 = 48px = 3rem
  • 16 = 64px = 4rem

Form Controls Spacing

Proper spacing between form elements improves readability and prevents visual "sticking"

Form Group Spacing

Use .mb-6 (24px) between form groups

<div class="form-group mb-6">
  <label class="label">Project Name</label>
  <input type="text" class="input">
</div>

<div class="form-group mb-6">
  <label class="label">Description</label>
  <textarea class="input"></textarea>
</div>

Checkbox & Radio Spacing

Use .mb-3 (12px) between individual controls

<!-- Vertical (block) layout -->
<label class="checkbox checkbox--block mb-3">
  <input type="checkbox" class="checkbox__input">
  <span class="checkbox__box">...</span>
  <span class="checkbox__label">Option 1</span>
</label>

<!-- Last item: NO mb-3 -->
<label class="checkbox checkbox--block">...</label>

<!-- Horizontal (inline) layout -->
<label class="checkbox">Option 1</label>
<label class="checkbox">Option 2</label>

Form Spacing Rules

  • Form Groups: Use .mb-6 (24px) between groups
  • Vertical Checkboxes/Radios: Add .checkbox--block modifier + .mb-3 (12px), except last
  • Horizontal Checkboxes/Radios: Default inline layout with automatic margin-left: 1.5rem (24px)
  • Toggle Switches: Use .mb-3 (12px) between items, except last
  • Within form-group: gap: 0.5rem (8px) between label and input
  • Submit Buttons: No margin-bottom on last form-group before buttons

Hierarchy

Golden Rules of Air

  • More air between sections - use .py-12 or .py-16
  • Medium air between groups - use .mb-6 or .mb-8
  • Less air inside a group - use .mb-3 or .mb-4
  • Minimum air between related elements - use .mb-2
  • Gap for layout - use gap instead of margin in grid/flex
