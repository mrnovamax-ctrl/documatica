üí° Usage Guidelines

  • Headings: Use semantic HTML (h1-h6) with .type-h* classes for styling
  • Display Text: For hero sections and landing pages, not for content headings
  • Body Text: Default is .type-body-base (16px), use others contextually
  • Labels: Always uppercase with wide letter-spacing for v12.0 style
  • Line Height: Headings tight (1.1-1.5), body generous (1.5-1.75)
  • Responsive: Display and heading sizes scale down on mobile automatically
