BACKGROUND PATTERNS

Background patterns for creating depth and texture in design. Documatica v12.0 uses dot grid and gradient patterns for hero sections and decorative elements.

01. Dot Grid Patterns

Dot grid for light and dark backgrounds

Light Dot Grid

Light background with blue dots. Use for hero sections and feature blocks.

.pattern-dots-light {
  background-color: #f8fafc;
  background-image: 
    radial-gradient(#3b82f6 1px, transparent 1px);
  background-size: 24px 24px;
}

Dark Dot Grid

Dark background with muted dots. Use for footer and dark mode sections.

.pattern-dots-dark {
  background-color: #0f172a;
  background-image: 
    radial-gradient(rgba(59, 130, 246, 0.3) 1px, transparent 1px);
  background-size: 24px 24px;
}

02. Grid Lines Pattern

Linear grid for structured sections

Grid Lines

Vertical and horizontal lines form a grid. Use for dashboard and data visualization sections.

.pattern-grid-lines {
  background-color: #f8fafc;
  background-image: 
    linear-gradient(rgba(59, 130, 246, 0.1) 1px, transparent 1px),
    linear-gradient(90deg, rgba(59, 130, 246, 0.1) 1px, transparent 1px);
  background-size: 24px 24px;
}

03. Radial Gradient Backgrounds

Radial gradients for hero sections and accents

Blue Radial Glow

Blue glow from center. Use for hero with CTA or feature highlights.

.pattern-radial-blue {
  background: 
    radial-gradient(
      circle at center, 
      rgba(59, 130, 246, 0.15) 0%, 
      transparent 70%
    );
}

Gold Radial Glow

Gold glow for AI features and premium elements.

.pattern-radial-gold {
  background: 
    radial-gradient(
      circle at center, 
      rgba(251, 191, 36, 0.15) 0%, 
      transparent 70%
    );
}

ðŸ’¡ Usage Guidelines

  • Dot Patterns: Use for hero sections and feature blocks. Don't overload the page - maximum 2-3 sections with patterns.
  • Radial Gradients: For accents and CTA blocks. Combine with dot pattern for depth.
  • Grid Lines: For dashboard, analytics, and data-heavy sections.
  • Solid Backgrounds: Alternate .bg-white and .bg-surface for visual separation of sections.
  • Dark Mode: Use .pattern-dots-dark and .bg-dark for dark sections (footer, modals).
  • Contrast: Always check text contrast on patterns - minimum 4.5:1 for body text, 3:1 for headings.
